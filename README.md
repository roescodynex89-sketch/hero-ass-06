# 🛒 DigiTools — Digital Tools Store

A modern digital products e-commerce store built with React 19, Vite, and Tailwind CSS. Browse digital tools, add them to your cart, and manage your selections with a smooth, responsive UI.

🔗 **Live Site:** https://radiant-torte-df16bf.netlify.app

📁 **Repository:** https://github.com/roescodynex89-sketch/hero-ass-06

## ✨ Features

- 🧭 **Navbar** — Cart icon with live item count
- 🎯 **Banner Section** — Hero section with heading, description, image, and CTA buttons
- 📊 **Stats Section** — Key product statistics display
- 🔄 **Product / Cart Toggle** — Switch between product listing and cart view
- 🃏 **Product Cards** — Name, description, price, period, tag, features, and icon
- 🛒 **Cart System** — Add/remove products, live count in navbar
- ✅ **Proceed to Checkout** — Clears all cart items on confirmation
- 🔔 **React Toastify** — Toast notifications for add, remove, and checkout actions
- 📱 **Fully Responsive** — Mobile and desktop friendly

## 🚀 Getting Started Locally

```bash
# 1. Clone the repository
git clone https://github.com/roescodynex89-sketch/hero-ass-06.git

# 2. Go into the project folder
cd hero-ass-06

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev

# 5. Open in browser
# http://localhost:5173
```
